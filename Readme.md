# Turbo Programming Language

## Supported Types
* `boolean`
* `integer`

## Operators
* `+, -, *, /`
* `- as prefix`

## Conditionals
```
if (<expression>) {
        // do something
    } else {
        // do something
}
```

## Variable bindings
```
let x = 1;
let y = 2;
let hawi = 420 * 1 + 2;
>> 422
```
